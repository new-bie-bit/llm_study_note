
## **1. Sigmoid 激活函数**
![Sigmoid](Static/Sigmoid.png)
### **定义与公式**

Sigmoid 函数是一个S形曲线函数，定义为：
$$
  \sigma(x) = \frac{1}{1 + e^{-x}}
$$
其输出范围在 (0, 1) 之间，常用于二分类问题的输出层。

导数公式：
$$
\sigma^{\prime}(z)=\sigma(z)(1-\sigma(z))
$$
### **特点与优缺点**

* **优点**：

  * 输出范围为 (0, 1)，非常适合用于概率输出。
  * 平滑且连续，适合用于很多机器学习模型，特别是二分类。
* **缺点**：

  * 梯度消失：当输入值很大或很小时，梯度趋近于零，导致网络学习变慢，尤其是在深层网络中。
  * 非零中心：输出值总是为正，这会影响模型的训练过程，特别是在权重更新时可能会造成不稳定。

### **图形展示**

```python
import numpy as np
import matplotlib.pyplot as plt

# Sigmoid function definition
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# Values for x from -10 to 10
x = np.linspace(-10, 10, 400)
y = sigmoid(x)

# Plotting the Sigmoid function
plt.plot(x, y, label='Sigmoid Activation')
plt.title('Sigmoid Activation Function')
plt.xlabel('Input (x)')
plt.ylabel('Output (σ(x))')
plt.grid(True)
plt.legend()
plt.show()
```

## **2. Tanh 激活函数**

### **定义与公式**

Tanh（双曲正切）是另一种常用的激活函数，其定义为：
[
\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}
]
其输出范围为 (-1, 1)，通常用于多层神经网络的隐层。

### **特点与优缺点**

* **优点**：

  * 输出是零中心的 (-1, 1)，能够加速优化过程。
  * 与 Sigmoid 类似，但其输出的零中心性质使得训练更加稳定。
* **缺点**：

  * 仍然存在梯度消失问题，当输入值很大或很小时，梯度变得接近零。

### **图形展示**

```python
# Tanh function definition
def tanh(x):
    return np.tanh(x)

# Values for x from -10 to 10
y_tanh = tanh(x)

# Plotting the Tanh function
plt.plot(x, y_tanh, label='Tanh Activation', color='orange')
plt.title('Tanh Activation Function')
plt.xlabel('Input (x)')
plt.ylabel('Output (tanh(x))')
plt.grid(True)
plt.legend()
plt.show()
```

## **3. ReLU 激活函数**

### **定义与公式**

ReLU（Rectified Linear Unit）是目前使用最广泛的激活函数之一，其定义为：
[
\text{ReLU}(x) = \max(0, x)
]
ReLU 只有在输入大于零时才会激活，如果输入小于零，输出为零。

### **特点与优缺点**

* **优点**：

  * 计算简单，高效。
  * 能有效避免梯度消失问题，特别是在正区间。
  * 在实践中，ReLU 的表现通常非常好，是深度学习中最常用的激活函数。
* **缺点**：

  * 死神经元问题：当输入小于零时，神经元输出为零，导致无法更新该神经元的权重。

### **图形展示**

```python
# ReLU function definition
def relu(x):
    return np.maximum(0, x)

# Values for x from -10 to 10
y_relu = relu(x)

# Plotting the ReLU function
plt.plot(x, y_relu, label='ReLU Activation', color='green')
plt.title('ReLU Activation Function')
plt.xlabel('Input (x)')
plt.ylabel('Output (ReLU(x))')
plt.grid(True)
plt.legend()
plt.show()
```

## **4. Leaky ReLU 激活函数**

### **定义与公式**

Leaky ReLU 是 ReLU 的一种变体，定义为：
[
\text{Leaky ReLU}(x) = \max(\alpha x, x)
]
其中 ( \alpha ) 是一个小常数（通常为 0.01）。

### **特点与优缺点**

* **优点**：

  * 解决了 ReLU 的死神经元问题，因为即使输入小于零，输出也不会完全为零，而是有一个很小的斜率（通常为0.01）。
* **缺点**：

  * 训练速度可能会比 ReLU 慢一些。

### **图形展示**

```python
# Leaky ReLU function definition
def leaky_relu(x, alpha=0.01):
    return np.maximum(alpha * x, x)

# Values for x from -10 to 10
y_leaky_relu = leaky_relu(x)

# Plotting the Leaky ReLU function
plt.plot(x, y_leaky_relu, label='Leaky ReLU Activation', color='red')
plt.title('Leaky ReLU Activation Function')
plt.xlabel('Input (x)')
plt.ylabel('Output (Leaky ReLU(x))')
plt.grid(True)
plt.legend()
plt.show()
```

## **5. ELU 激活函数**

### **定义与公式**

ELU（Exponential Linear Unit）是 ReLU 的另一种改进版本，定义为：
[
\text{ELU}(x) = \begin{cases}
x, & \text{if } x \geq 0 \
\alpha(e^x - 1), & \text{if } x < 0
\end{cases}
]
其中 ( \alpha ) 是一个超参数，通常设为 1。

### **特点与优缺点**

* **优点**：

  * 具有平滑的负区间输出，能够更好地捕捉负值特征，避免死神经元问题。
  * 对于负输入，能够以指数形式增长，提高了模型的非线性。
* **缺点**：

  * 计算复杂度较高，可能会影响训练速度。

### **图形展示**

```python
# ELU function definition
def elu(x, alpha=1):
    return np.where(x >= 0, x, alpha * (np.exp(x) - 1))

# Values for x from -10 to 10
y_elu = elu(x)

# Plotting the ELU function
plt.plot(x, y_elu, label='ELU Activation', color='purple')
plt.title('ELU Activation Function')
plt.xlabel('Input (x)')
plt.ylabel('Output (ELU(x))')
plt.grid(True)
plt.legend()
plt.show()
```

## **6. Softmax 激活函数**

### **定义与公式**

Softmax 是多分类问题中常用的激活函数，它将网络的输出转换为概率分布，定义为：
[
\text{Softmax}(x_i) = \frac{e^{x_i}}{\sum_{j} e^{x_j}}
]
其中 (x_i) 是网络输出的某一项。

### **特点与优缺点**

* **优点**：

  * 将输出值转换为概率分布，便于分类问题的处理。
  * 确保所有输出的和为 1。
* **缺点**：

  * 计算量较大，尤其是在类别数很多时。

### **图形展示**

Softmax 的输出是一个概率分布，通常需要多个输出才能展示其特点。我们可以通过一组输入数据来展示 Softmax 的行为。

```python
# Softmax function definition
def softmax(x):
    exp_x = np.exp(x - np.max(x))  # Subtracting max for numerical stability
    return exp_x / np.sum(exp_x)

# Sample values for x
x_softmax = np.array([1.0, 2.0, 3.0, 4.0])
y_softmax = softmax(x_softmax)

# Plotting the Softmax function output
plt.bar(range(len(x_softmax)), y_softmax, tick_label=["Class 1", "Class 2", "Class 3", "Class 4"])
plt.title('Softmax Activation Function Output')
plt.ylabel('Probability')
plt.xlabel('Class')
plt.show()
```
